para el despliegue de la app de manera local debe de abrirse la aplicación XAMPP, en en archivo htdocs debe incluirse la carpeta completa de la app, en el administrador de sql de XAMPP debe crearse una bd con el nombre rf y dentro de ella i importar el documento que contiene la estructura de bases de datos y los datos de prueba, una vez lanzados el servidor apache y la base de datos accediendo a localhost/pirf-app entraremos en la app
